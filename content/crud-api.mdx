## Introduction

If you’ve ever worked with APIs, you’ve already used CRUD — even if you didn’t realize it.

CRUD is the foundation of almost every modern application. Whether you're building a blog, an e-commerce platform, or a SaaS product, everything revolves around creating, reading, updating, and deleting data.

In this guide, you’ll learn CRUD in a way that actually makes sense — with real examples and clear explanations.

---

## What is CRUD?

CRUD stands for:

- Create  
- Read  
- Update  
- Delete  

These four operations define how you interact with data.

Think of it like managing users in an app:

- Create → Add a new user  
- Read → Get user details  
- Update → Edit user info  
- Delete → Remove user  

---

## HTTP Methods Mapping

Each CRUD operation maps to an HTTP method:

| Operation | Method | Description |
|----------|--------|-------------|
| Create | POST | Add new data |
| Read | GET | Fetch data |
| Update | PUT | Modify existing data |
| Delete | DELETE | Remove data |
